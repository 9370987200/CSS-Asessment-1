  2. CSS describes how HTML elements are to be dispayed on screen,paper,or in other media.
  3. Css is not neccesary CSS saves a lot of work. It can control the layout of mutiple web pages at once

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements 1
  • Internal - by using a style element in the head section
  • External - by using a link to an external CSS file
  • The most common to add CSS , is to keep the styles in external css files . However , in this tutorial we will use inline and internal styles , because this is easier to demonstrate , and easier for you to try it yourself.
